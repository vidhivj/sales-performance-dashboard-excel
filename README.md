# Sales Performance Dashboard (Microsoft Excel)

## Overview
This project is an interactive Sales Performance Dashboard created in Microsoft Excel using Pivot Tables, Pivot Charts, Formulas, and Dashboard Design techniques.

## Features
- KPI Cards
- Sales Trend Analysis
- Region-wise Sales Analysis
- Product-wise Sales
- Salesperson Performance
- Dashboard Insights
- Navigation Buttons
- Interactive Pivot Tables

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Formulas
- Conditional Formatting
